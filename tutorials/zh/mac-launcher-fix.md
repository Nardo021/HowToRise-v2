# 修复 macOS 新版 Rise 启动器问题

如果你在 macOS 上使用**新版 Rise 启动器**时遇到问题，重置其数据可以解决大多数问题，包括某些 JRE/JDK 崩溃。

## 删除 Rise 启动器文件夹

1. 打开 **Finder**
2. 按 `Cmd + Shift + G`
3. 前往：

   ```
   ~/Library/Application Support/
   ```
4. 找到 **Rise** 文件夹并删除。

## 此修复的作用

* 重置启动器数据和缓存文件
* 修复大多数与新版启动器更新相关的问题

> 注意：目前只有新版启动器包含最新更新。
