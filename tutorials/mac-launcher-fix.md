# Fix Issues with the New Rise Launcher (macOS)

If you’re experiencing problems with the **new Rise Launcher** on macOS, resetting its data may resolve most issues, including certain JRE/JDK crashes.

## Delete the Rise Launcher Folder

1. Open **Finder**
2. Press `Cmd + Shift + G`
3. Go to:

   ```
   ~/Library/Application Support/
   ```
4. Find the **Rise** folder and delete it.

## What This Fix Does

* Resets launcher data and cached files
* Fixes most issues related to the new launcher update

> Note: Currently, only the new launcher includes the latest update.
